**KidsRiseNShine** is a vibrant, interactive web application designed to help parents and children manage morning routines effectively. By gamifying daily tasks, the app encourages independence in children and reduces morning stress for the whole family.

---

## üöÄ Overview

Morning routines can be a struggle. **KidsRiseNShine** transforms chores and "getting ready" tasks into a fun, rewarding experience. Children can track their progress through a visual interface, while parents can customize schedules to fit their family's specific needs.

## ‚ú® Key Features

* **Interactive Checklists:** Simple, icon-based tasks (brushing teeth, making the bed, packing bags) that kids can check off themselves.
* **Reward System:** Earn points or digital "stars" for completing tasks on time to unlock rewards.
* **Parental Dashboard:** Easy-to-use interface for parents to add, edit, or remove tasks and monitor progress.
* **Visual Timers:** Helps children understand how much time they have left for specific activities.
* **Responsive Design:** Fully functional on desktops, tablets, and smartphones.

## üì¶ Installation & Setup

To get a local copy up and running, follow these steps:

### Prerequisites

* Node.js (v18.0.0 or higher)
* npm or yarn

### Steps

1. **Clone the repository:**
```bash
git clone https://github.com/NatalieM82/KidsRiseNShine.git

```


2. **Navigate to the project directory:**
```bash
cd KidsRiseNShine

```


3. **Install dependencies:**
```bash
npm install

```


4. **Start the development server:**
```bash
npm start

```


The application will be available at `http://localhost:3000`.

## üñ•Ô∏è Usage

1. **Setup Tasks:** Parents enter the "Settings" or "Parent Mode" to define the morning routine.
2. **Morning Mode:** Switch to the "Child View" and place the device in a shared area (like the kitchen or bathroom).
3. **Complete & Celebrate:** As the child clicks tasks, animations provide instant positive reinforcement.
